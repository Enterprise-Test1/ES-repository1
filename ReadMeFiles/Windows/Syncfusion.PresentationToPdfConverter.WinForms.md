### Getting started
You can fetch the Syncfusion .NET PowerPoint library NuGet by simply running the command `Install-Package Syncfusion.PresentationToPDFConverter.WinForms` from the Package Manager Console in Visual Studio.

Try the following code example to convert the PowerPoint Presentation to PDF. 

```csharp
//Namespaces to perform PPTX to PDF conversion
using Syncfusion.Presentation;
using Syncfusion.OfficeChartToImageConverter;
using Syncfusion.PresentationToPdfConverter;
using Syncfusion.Pdf;

//Opens a PowerPoint Presentation 
using (IPresentation pptxDoc = Presentation.Open("Sample.pptx"))
{
	//Creates an instance of ChartToImageConverter and assigns it to ChartToImageConverter property of Presentation
	pptxDoc.ChartToImageConverter = new ChartToImageConverter();
	//Convert the PowerPoint document to PDF document.
	using (PdfDocument pdfDocument = PresentationToPdfConverter.Convert(pptxDoc))
	{
		//Creates an instance of memory stream.
		using (MemoryStream pdfStream = new MemoryStream())
		{
		  //Save the converted PDF document to memory stream.
		  pdfDocument.Save(pdfStream);
		}
	}
}
```

For more information to get started, refer to our [Getting Started Documentation page](https://help.syncfusion.com/file-formats/presentation/create-read-edit-powerpoint-files-in-winforms?utm_source=nuget&utm_medium=listing&utm_campaign=winforms-presentationtopdfconverter-nuget).
